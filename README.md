An IP tracker React web app that shows the location of the given IP input by user using Leaflet JS and ipinfo.io API
This is a fully responsive IP tracker React app displays the user's IP geolocation on initialization of the page. Users may input an IP address and it moves the map accordingly with a neat loading interface.
